Analysis

We will perform Latent Dirichlet Allocation (LDA) Ideally, since we are going to use the founds topics for similarity analysis, we should use a correlated topic model (CTM). I could not find a working implementation of that and didn't have the time to fix one, so LDA will have to do. We can estimate topic correlations adhoc using either ϕϕ' (on a word similarity basis) or θθ' (on author similarity basis)

LDA

function corr(x::AbstractMatrix)
    d = diag(x)
    y = copy(x)
    for i = 1:length(d), j = 1:length(d)
        y[i,j] /= sqrt(x[i,i]*x[j,j])
    end
    y
end

The current staff at the department is listed in staff.txt Since the names of the authors are not on the same format in the database and on the webpage, we use a string comparison tool to find correspondences. "The distance Tokenmax(RatcliffObershelp()) is a good choice to link names or adresses across datasets."

staff = readlines("staff.txt")

staff2unique_author = map(staff) do staffmember
    distances = map(unique_authors) do author
        StringDistances.compare(TokenMax(RatcliffObershelp()), filter(isascii,author), filter(isascii,staffmember))
    end
    max_i = argmax(distances)
    distances[max_i] < 0.9 && return 0
    return max_i
end
filter!(!iszero, staff2unique_author)
authornames = unique_authors[staff2unique_author]

To find the topic vectors of each staff members, we average over all their publications. This is a simple way of doing things, but might cause a senior author with a diverse set of publications to appear as having little similarity with a young researcher with a narrow focus, even if the senior author has a few publications in that particular topic.

staffvecs = map(authornames) do staffmember
    r = Regex(staffmember)
    authored = map(authors) do author
        match(r, author) != nothing
    end
    authorinds = findall(authored)

We can also analyze the cov between authors Before we plot the cov matrix, we try to approximately diagonalize it using the AMD algorithm. To do this, we have to set some elements that fall beneath a certain threshold to zero. We implement a simple heuristic to set this tolerance automatically

function diagonalize(C, tol=:auto; permute_y=false, doplot=false)
    C = copy(C)
    amdmat = size(C,1) == size(C,2) ? copy(C) : C'C
    # amdmat = C'C
    if tol == :auto # Find tolerance automatically (adhoc)
        y    = abs.(amdmat[:])
        inds = sortperm(y)
        cum  = cumsum(y[inds])
        cum ./= cum[end]
        i    = findfirst(c->c > 0.3, cum)
        tol  = y[inds[i]]
    end
    doplot && (histogram(abs.(amdmat[:])) |> display)
    amdmat[abs.(amdmat) .< tol] .= 0
    perm  = amd(sparse(amdmat))
    yperm = permute_y ? perm : 1:size(C,1)
    C[yperm,perm], perm, yperm
end
function plotcov(C, xvector, yvector; kwargs...)
    xticks = (1:length(xvector), xvector)
    yticks = (1:length(yvector), yvector)
    heatmap(C; xticks=xticks, yticks=yticks, xrotation=50, title="Author similarity", kwargs...)
end
